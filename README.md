# Español

## Un beat. Todas las plataformas.

Tu beat ya está terminado.

El trabajo todavía no.

Ahora necesitas:

* Un máster limpio en WAV y MP3
* Versiones etiquetadas para previews y tiendas
* Vídeos de YouTube con visualizadores
* Vídeos verticales para TikTok, Shorts y Reels
* Contenido cuadrado para redes sociales
* Portadas y miniaturas
* Títulos, descripciones, hashtags y metadatos
* Un paquete organizado con todos los archivos finales

Hacer todo esto manualmente puede llevar horas por cada beat.

**DIAVLO DROP convierte todo ese proceso en un solo flujo.**

> Suelta el beat. Construye el lanzamiento. Exporta todo.

## Un archivo. Un drop completo.

```text
                           TU BEAT
                              │
                              ▼
                     ┌─────────────────┐
                     │   DIAVLO DROP   │
                     └─────────────────┘
                              │
           ┌──────────────────┼──────────────────┐
           ▼                  ▼                  ▼
         AUDIO             VISUALES          METADATOS
           │                  │                  │
      Máster WAV          Portada          Texto YouTube
      MP3 320 kbps        Miniatura        Texto TikTok

      Preview 60 s        Vídeo cuadrado   Hashtags
           │                  │                  │
           └──────────────────┼──────────────────┘
                              ▼
                     PAQUETE FINAL COMPLETO
```

## Flujo de trabajo

1. Abre DIAVLO DROP.
2. Completa la configuración inicial.
3. Elige tu nombre de productor y carpeta de salida.
4. Arrastra un beat a la biblioteca.
5. Revisa o modifica el BPM y la tonalidad.
6. Edita el título, género y metadatos.
7. Añade un producer tag si lo deseas.
8. Crea una portada usando las plantillas integradas.
9. Configura el visualizador.
10. Elige un preset de exportación.
11. Añade el beat a la cola.
12. Obtén el paquete final en tu carpeta de salida.

## Workflow

1. Open DIAVLO DROP.
2. Complete the initial setup.
3. Choose your producer name and output folder.
4. Drop a beat into the library.
5. Review or edit the detected BPM and key.
6. Edit the title, genre and metadata.
7. Add a producer tag if needed.
8. Create artwork using the built-in templates.
9. Configure a visualizer.
10. Select an export preset.
11. Add the beat to the queue.
12. Get the complete package in your output folder.
